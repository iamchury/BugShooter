## BugShooter v1.0

First playable release.

### Features

- Portrait-mode Android shooting game
- Full 2D aircraft movement
- Automatic firing while touching and dragging
- Ten-mosquito formation groups
- Multiple randomized flight patterns
- Explosion and player-hit effects
- Score penalty when the aircraft is hit
- Responsive screen layout

### Installation

1. Download `BugShooter-v1.0.apk`.
2. Allow installation from unknown apps when Android requests it.
3. Install and run the APK.
